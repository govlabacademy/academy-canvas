The GovLab Academy Canvas - 0.2 - Release Notes

Version 0.2

The GovLab Academy Canvas outlines 20 questions to be answered in the course of designing an actionable social change project.

Requirements: A modern browser. This prototype is created using chrome. It should work just fine in Safari and Firefox too.

Release Updates:

  • Added Description fields (One Liner, Authors, Location fields)
  • Last Edited Feature - now each question displays how long ago it was edited. It also indicates if it was an answer or a comment.
  • Password updates - now users can add/edit/remove a password from any canvas using the cog icon on the top bar.
